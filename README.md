A microservice-based real-time system simulating smart city sensor data (traffic, weather, air quality, etc.), streaming it through Kafka to analytic services that store, visualize, and run ML predictions (e.g., predict pollution or traffic levels).

Real-time data streaming with Kafka

Microservices (Node.js, Python, FastAPI)

ML model prediction integrated into the pipeline

Containerization (Docker)

Orchestration (Kubernetes)

Logging/Monitoring (Prometheus + Grafana)

CI/CD pipeline (GitHub Actions)
